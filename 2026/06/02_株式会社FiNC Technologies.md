# 本文
平素よりFiNC Uをご利用いただき、誠にありがとうございます。

このたび、弊社が運営するオウンドメディア「FiNC U」の一部ページにて、5月31日夜間から6月1日15時頃にかけて、外部ライブラリ（polyfill.io）の改ざんに起因する不審なログイン画面が表示される問題が発生していたことを確認いたしました。

現在は当該外部サービスとの連携を解除し、再発防止に向けた対応を完了しております。

つきましては、上記の期間中に表示された画面でユーザー名・パスワードを入力された方は、同じ認証情報を他のサービスでも使い回している場合を含め、早急にパスワードの変更をお願いいたします。

現時点では、FiNC Uのシステムへの不正アクセスや、お客様情報の漏洩は確認されておりません。ただし、念のためご確認・ご対応いただけますと幸いです。

このたびはご不便・ご心配をおかけしましたことを心よりお詫び申し上げます。

引き続きFiNC Uをご愛顧いただければ幸いです。
