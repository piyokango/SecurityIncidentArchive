# 本文
このたび、当財団が運営する「優良さんぱいナビ」に対し外部からの不正アクセスが発生し、「優良さんぱいナビ」に保存されていたシステム利用者に関する情報等の一部が外部へ漏えいした可能性の高いことがわかりました。

システムをご利用いただいている皆様に多大なるご迷惑とご心配をおかけする事態となりましたことを、心より深くお詫び申し上げます。

## 1. 事案の概要
令和８年５月 25 日、「優良さんぱいナビ」及び「さんぱいくん」においてデータベースの異常な高負荷状態を検知したため、直ちに調査を行いました。

当該調査の結果、第三者による不正アクセスが確認され、さらに精査したところ、システム内の情報の一部が外部へ漏えいした可能性の高いことがわかりました。

## 2. 漏えいした可能性の高い情報
漏えいした可能性の高い情報は以下のとおりです。
- 入力担当者氏名
- ID
- パスワード
- メールアドレス
※現時点においては、本件に起因する不正利用等の具体的な被害は確認されておりませんが、引き続き監視および調査を継続しております。

## 3. 原因
本件は外部からの不正アクセスによるもので、海外からの攻撃であることが判明しています。

## 4. 現在までの対応
５月 25 日から以下の対応を実施いたしました。
- 対象システムの停止
- 外部接続の遮断

## 5. 今後の対応および再発防止策
再発防止に向けシステムの脆弱性対応の強化策を講じてまいります。

## 6. システム利用者の皆様へのお願い
大変恐縮ではございますが、以下の点についてご対応をお願い申し上げます。
- 本サービスで使用されていたパスワードを他のサービスでも使用されている場合には、当該サービスの不正利用防止の観点から、速やかに変更いただきますようお願いいたします。
- 本件により、利用者様の氏名やメールアドレスが漏えいしている可能性があることから、これらの情報を悪用したなりすまし等の不正行為が行われるおそれがあります。当財団を装った不審なメールや連絡には十分ご注意ください。
- 万が一、不審な事象を確認された場合には、速やかに下記窓口までご連絡いただきますようお願いいたします。
