# 本文
平素よりアソビューをご利用いただき、誠にありがとうございます。

この度、弊社がお取引先様（以下「パートナー様」）に提供するシステムへのサイバー攻撃により、パートナー様情報の一部が外部に流出したことを確認いたしましたので、判明した事実ならびに弊社の対応について、下記の通りご報告申し上げます。パートナー様・お客様をはじめ、関係者の皆さまに多大なるご迷惑、ご心配をおかけしておりますことを深くお詫び申し上げます。

なお、すでに内部での調査を終え、原因と影響範囲の特定を完了しております。これに伴う短期的な防衛対策は既に実施済みであり、さらなる追加の対策についても順次実施していく方針で進めております。

## 1. 影響の範囲
本件の対象となるパートナー様には、影響範囲の詳細を個別にご連絡しております。個別メールが届いていないパートナー様におかれましては、今回の対象ではございませんのでご安心ください。

|対象者|内容|件数|
|---|---|---|
|satsuki経由の不正ログインが確認されたパートナー様|・会社名、住所、電話番号<br/>・代表者氏名、請求担当者氏名、請求先住所、口座情報、請求書、支払通知書<br/>・各種メールアドレス|111件|
|その他のパートナー様|・店舗名、住所、電話番号<br/>・請求担当者氏名、請求先住所、口座情報<br/>・各種メールアドレス|14,400件|

なお、お客様（ゲスト様）の情報流出の範囲および詳細につきましてはこちらの[お客さま向けお知らせページ］をご確認ください。

## 2. 経緯および詳細
2026年5月20日に弊社が提供するアクティビティ事業者向け予約管理システム「satsuki」（以下「satsuki」）に対し、外部からの不審なアクセスを検知いたしました。ただちに調査を開始し、不正アクセス元および類似のアクセスパターンのブロック措置を実施し、監視を継続しております。

現在までの調査の結果、サイバー攻撃による不正アクセスであると判断しました。現時点で判明している不正アクセスの内容は以下の通りとなります。

- satsukiに対する一部パートナー様アカウントへの不正ログイン
- 上記不正ログイン後における、当該パートナー様の予約者情報（ゲスト情報）に対するアクセス
- 上記不正ログイン後、認証情報の悪用による当該パートナー様以外のパートナー様情報に対するアクセス

現時点で、流出した情報の悪用による被害の発生は確認されておりません。

また、弊社サポートデスクへの上記に該当する相談、報告は受けておりません。

しかしながら、今回は各種メールアドレスや口座情報等も対象に含まれているため、今後、流出した情報を悪用した「なりすましメール」や「フィッシングメール（偽の請求・支払い案内等含む）」が送付される可能性を完全に否定することはできません。

万が一、不審なメールや添付ファイルが送られてきた場合は、開封せず削除するなど、十分にご注意いただきますようお願い申し上げます。

なお、弊社は本件について2026年5月21日付で個人情報保護委員会へ報告、および警察当局へ相談し、被害届提出の準備を進めております。

## 3. 現時点で実施している対応について
弊社では、不正アクセス検知後、速やかに防御・監視の強化等の対応を行ってまいりました。

その一環として、5月20日にsatsukiの全アカウントに対するパスワードの強制リセットを実施しております。また、5月26日からはセキュリティ強化の一環として、パートナー様が利用する他サービス（ウラカタシリーズ等）についてもパスワードリセットを実施しております。

お手数をおかけし恐縮ではございますが、安全のため、パスワードの再設定にご協力をお願いいたします。

## 4. 今後の再発防止策およびご報告について
今後のサイバー攻撃を回避するための追加の対策・再発防止策については、すでに洗い出しが完了しております。

併せて専門の第三者機関とも連携し、客観的な調査と追加対策の特定も進めております。

これらの対応の進捗や詳細などについては、続報としてパートナー様各社に改めてご報告いたしますが、攻撃手口の模倣や追加攻撃を含む二次被害防止の観点から、技術的な詳細の開示につきましては差し控えさせていただく場合もございますので、予めご了承ください。
