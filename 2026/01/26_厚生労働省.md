# 公表概要
- (お知らせ) Web サイト「あかるい職場応援団」改ざんに関するご報告とお詫び
- 2026年1月26日
- 厚生労働省
- https://www.no-harassment.mhlw.go.jp/pdf/20260126_news.pdf

# 本文
今般、第三者により、厚生労働省が運営する「あかるい職場応援団」Web サイトが改ざんされ、本来とは異なるページが表示されていたことが判明いたしました。

本件について、深くお詫びいたしますとともに、以下のとおり、概要についてご報告いたします。

現在、本改ざんについては既に対策を講じた上、同 Web サイトを復旧しております。

また、現時点において、個人情報等の流出はなく、不適切なページが表示されたことによる被害等について、利用された方々からのご連絡もいただいておりません。

厚生労働省で把握している情報は以下のとおりとなっております。

改ざん期間：令和８年１月 13 日(火) 13 時 14 分～１月 14 日（水）14 時 03 分

この期間にアクセスされた可能性のある利用者様におかれましては、誠にお手数ですが、お手持ちのセキュリティソフトを最新の状態にし、ウイルスチェック・駆除の実施をお願いいたします。

今後、このような問題が発生しないよう、より一層の注意を払い、同サイトの管理を実施してまいります。

本件につきましては、ご不明な点がございましたら、下記までご連絡くださいますようお願い申し上げます。
